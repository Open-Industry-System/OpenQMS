## 数据模型

### Alembic 迁移（单次 migration）

```sql
-- sample_batches 扩展
ALTER TABLE sample_batches
  ADD COLUMN inspected_count INTEGER,   -- 本批次检验总数 n（P/NP/C/U 使用，计量值图为 NULL）
  ADD COLUMN defect_count    INTEGER;   -- 不合格品数（P/NP）或缺陷数（C/U），计量值图为 NULL

-- control_limit_snapshots 扩展
ALTER TABLE control_limit_snapshots
  ADD COLUMN version_no  INTEGER NOT NULL DEFAULT 1,
  ADD COLUMN is_active   BOOLEAN NOT NULL DEFAULT false;
```

### 图表类型扩展

| chart_type | 中文名 | 统计量 | 控制限公式 | 控制线形态 |
|---|---|---|---|---|
| p  | 不合格率图 | pi = defect/inspected | p_bar ± 3√(p_bar(1-p_bar)/ni) | UCL/LCL 折线 |
| np | 不合格品数图 | defect_count | np_bar ± 3√(np_bar(1-p_bar)) | UCL/LCL 水平线 |
| c  | 缺陷数图 | defect_count | c_bar ± 3√c_bar | UCL/LCL 水平线 |
| u  | 单位缺陷数图 | ui = defect/inspected | u_bar ± 3√(u_bar/ni) | UCL/LCL 折线 |

**判异规则**：计数值图仅启用规则1（超出控制限）。  
**LCL 截断**：负值截断为 0。

### 版本管理逻辑

- 每次锁定控制限 → version_no = MAX(已有)+1，is_active=true，其余置 false
- PATCH activate → 切换 is_active
- 版本只增不减

## 计算引擎

文件：backend/app/services/spc_calculation_engine.py

新增4个函数：
- calculate_p_limits(batches) → {cl, ucl_list, lcl_list}（变样本量）
- calculate_np_limits(batches) → {cl, ucl, lcl}（固定n，不一致时 raise ValueError）
- calculate_c_limits(batches) → {cl, ucl, lcl}
- calculate_u_limits(batches) → {cl, ucl_list, lcl_list}（变样本量）

## API

### 现有接口变更

- POST /api/spc/characteristics：chart_type 校验扩展至 p|np|c|u
- POST /api/spc/characteristics/{id}/batches：新增可选 inspected_count、defect_count
- POST /api/spc/characteristics/{id}/lock-limits：响应新增 version_no
- GET /api/spc/characteristics/{id}/chart：响应新增 active_snapshot 字段

### 新增接口

- GET /api/spc/characteristics/{id}/snapshots：版本列表，按 version_no 降序
- PATCH /api/spc/characteristics/{id}/snapshots/{sid}/activate：切换 active 版本

## 前端

### 新建特性弹窗

- chart_type 下拉增加 P/NP/C/U 四个选项
- 计数值图时隐藏 subgroup_size 字段
- NP/C 图显示"样本量需固定"提示

### 录入批次弹窗

- 计量值图：现有测量值输入，不变
- 计数值图：替换为"检验数量 n"和"不合格品数/缺陷数"两个整数输入框

### 控制图渲染

- P/U 图：UCL/LCL 渲染为折线（来自 ucl_list/lcl_list 数组）
- NP/C 图：UCL/LCL/CL 均为水平线
- 计数值图下隐藏能力分析面板

### 版本管理面板（VersionPanel）

- 详情页右上角"版本管理"按钮 → Ant Design Drawer
- 列表：v{n} | 日期时间 | [设为当前]，active 版本高亮
- 点击"设为当前" → PATCH activate → 图表重载
- 控制图标题旁显示"控制限 v{n}"徽标

---

## 约束与边界

- NP 图 inspected_count 不一致 → 后端 ValueError → 前端提示
- LCL 负值截断为 0
- 版本切换不影响 SampleBatch 数据
- 计数值图不显示 Cp/Cpk 面板
- 版本不支持删除

---

## 不在范围内

- 版本间差异对比视图
